# Baseline behavior (Phase 1 — frozen)

The **first acceptance criterion** of the cross-DAW program is: *the existing
Logic explorer still works.* This document records the behavior that must remain
true after every subsequent phase, and the golden fixture that guards it.

## Reproduce the baseline

```bash
pip install -r requirements.txt
# CLI, fully deterministic (no audio analysis):
python -m logic_session_evidence_explorer demo --no-descriptors
# App:
streamlit run src/logic_session_evidence_explorer/app.py   # Built-in demo → Build demo session
```

## Golden snapshot

`fixtures/baseline/demo_full_bundle.json` and `demo_graph.json` are the
**deterministic** (no-descriptors) export of the built-in demo, committed as a
regression anchor. They are regenerated by:

```bash
PYTHONPATH=src python3 -c "from logic_session_evidence_explorer import demo, export, utils; \
s=demo.build_demo_session(with_descriptors=False); \
utils.write_json('fixtures/baseline/demo_full_bundle.json', export.full_bundle(s)); \
utils.write_json('fixtures/baseline/demo_graph.json', export.graph_json(s))"
```

The regression test `tests/test_baseline.py` asserts the live demo build still
matches this committed structure.

## Invariants that must not regress

For the built-in demo, **no-descriptors** path (deterministic):

| Property | Value |
| --- | --- |
| `session_name` | `Logic Indie Mix Evidence Demo` |
| `source_type` | `synthetic_demo` |
| audio files / inferred tracks | 7 / 6 |
| hidden-state markers | 6 |
| graph nodes / edges | 36 / 35 |
| observability split | observed 22.2% · inferred 16.7% · annotation 38.9% · hidden 16.7% · derived 5.6% |
| bundle `export_metadata.native_project_parsed` | `false` |

With descriptors enabled (environment-dependent, **not** frozen): +descriptor
and stem-sum-reconciliation nodes, 3 recommendations, `native_project_parsed`
still `false`.

## What must keep working (behavioral)

1. The four app modes render; **Build demo session** produces the summary,
   evidence meter, layered graph, recommendations, and exports with no
   exception.
2. Role inference, hidden-state markers, recommendations, PROV-O export, the
   observability meter, and the benchmark (`eval/`, 99.3% in-sample) are
   unchanged unless a phase explicitly and verifiably improves them.
3. The full test suite (currently **112**) stays green on Linux CI (Python
   3.10–3.12). Any live/native capture added later must degrade to
   `unsupported` off-macOS so this holds.

## Not committed (data safety)

No real audio, no `.logicx` bundles, no absolute user paths, no credentials. The
demo generates synthetic audio on the fly; fixtures are structural JSON only.
